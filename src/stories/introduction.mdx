# @pulsar-framework/formular-ui

Welcome to the **Pulsar Formular UI** component library!

## Overview

This library provides a comprehensive set of form components built on top of:

- **Pulsar Framework**: Reactive UI framework with fine-grained reactivity
- **formular.dev**: Modern form builder with validation
- **Tailwind CSS**: Utility-first CSS framework
- **Design Tokens**: Shared design system from @pulsar-framework/design-tokens

## Features

- ✅ Fully reactive form components
- ✅ Built-in validation
- ✅ Accessible by default
- ✅ Type-safe with TypeScript
- ✅ Customizable with Tailwind CSS
- ✅ Storybook documentation

## Getting Started

Browse the component stories in the sidebar to see examples and documentation for each component.

### Installation

```bash
npm install @pulsar-framework/formular-ui
```

### Basic Usage

```typescript
import { FormInput, Form } from '@pulsar-framework/formular-ui';

// Your form implementation
```

## Component Categories

- **Basic Inputs**: Text, Email, Password, Number
- **Selection**: Select, Radio, Checkbox
- **Text**: Textarea
- **Layout**: Form, FormGroup, FormField
